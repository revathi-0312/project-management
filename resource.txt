Enhance your existing database to include resource-related tables:

sql
Copy
Edit
CREATE TABLE resources (
    resource_id INT AUTO_INCREMENT PRIMARY KEY,
    resource_name VARCHAR(255) NOT NULL,
    resource_type ENUM('Human', 'Material', 'Cost') NOT NULL,
    project_id INT,
    allocation_hours INT,
    cost DECIMAL(10, 2),
    FOREIGN KEY (project_id) REFERENCES projects(project_id)
);
This schema allows you to associate resources with specific projects and track their allocation and costs.

b. Backend Logic
Modify your backend logic to handle resource assignments:

php
Copy
Edit
// Assign a resource to a project
$resource_name = 'John Doe';
$resource_type = 'Human';
$project_id = 1;
$allocation_hours = 160;
$cost = 2000.00;

$stmt = $conn->prepare("INSERT INTO resources (resource_name, resource_type, project_id, allocation_hours, cost) VALUES (?, ?, ?, ?, ?)");
$stmt->bind_param("ssiid", $resource_name, $resource_type, $project_id, $allocation_hours, $cost);
$stmt->execute();
This code snippet demonstrates how to assign a human resource to a project with specified hours and cost.

    <label for="cost">Cost:</label>
    <input type="number" id="cost" name="cost" step="0.01" required>

    <button type="submit">Assign Resource</button>
</form>
This form allows users to assign resources to projects by specifying the resource name, type, project ID, allocation hours, and cost.
